Send to W2G Settings

Get your API key from W2G account settings
If provided, tries to use existing room (must be member)

How to use this extension:

  1. Get your API key from Watch2Gether account settings
  2. Save your API key in this extension
  3. Click "Send to W2G" on any YouTube video
  4. The extension will create a new room or use your existing one

Note: If you provide a room key, you must be a member of that room for the API to work.
